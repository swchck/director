# 07 — Schema as Code

Demonstrates creating Directus collections, fields, relations, and flows programmatically.

**What you'll learn:**
- Creating collections with proper UI fields (not "Database Only")
- M2O relations (product → category dropdown)
- M2M relations via junction tables (products ↔ tags)
- Translation collections
- Collection folders for sidebar organization
- Automation flows with operations

```bash
task up
go run example/07-schema-as-code/main.go
```

See the code for the full e-commerce schema setup.
